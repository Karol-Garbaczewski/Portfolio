## Project overview
This project transforms raw data from my Dungeons & Dragons campaign into a Business Inteligence dashboard. It demonstrates the end-to-end process of data gathering, preparation, modeling, and visualization. 

## Key features
* **Microsoft Power BI:** Data modeling, DAX, and report design.
* **Microsoft Excel:** Data entry and initial data structuring.
## Data Model & Calculations:
* The project utilizes a relational model where session data is linked to character profiles. 
* I introduced the DAX measure that shows the luckiness of each character.
